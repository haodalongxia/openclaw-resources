# 第 1 章：Claude 基础入门

## 1.1 Claude 是什么

Claude 是 Anthropic 公司开发的人工智能助手，基于大型语言模型（LLM）技术。

### 核心能力
- 自然语言对话
- 代码生成和调试
- 文档写作和编辑
- 数据分析和总结
- 多语言翻译

### 与其他 AI 的区别
| 特性 | Claude | GPT-4 | Gemini |
|------|--------|-------|--------|
| 上下文长度 | 200K tokens | 128K tokens | 100K tokens |
| 安全性 | 高（Constitutional AI） | 中 | 中 |
| 代码能力 | 强 | 强 | 中 |
| 中文支持 | 好 | 好 | 好 |

## 2.2 第一个 AI 项目

### 项目：智能待办清单

**功能需求**:
- 添加任务
- 删除任务
- 标记完成
- 列表显示

**代码实现**:
```python
class TodoList:
    def __init__(self):
        self.tasks = []
    
    def add(self, task):
        self.tasks.append({"task": task, "done": False})
        return f"已添加：{task}"
    
    def delete(self, index):
        if 0 <= index < len(self.tasks):
            removed = self.tasks.pop(index)
            return f"已删除：{removed['task']}"
        return "索引超出范围"
    
    def complete(self, index):
        if 0 <= index < len(self.tasks):
            self.tasks[index]["done"] = True
            return f"已完成：{self.tasks[index]['task']}"
        return "索引超出范围"
    
    def list(self):
        if not self.tasks:
            return "暂无任务"
        result = []
        for i, t in enumerate(self.tasks):
            status = "✓" if t["done"] else "○"
            result.append(f"{i}. [{status}] {t['task']}")
        return "\n".join(result)

# 使用示例
todo = TodoList()
print(todo.add("学习 Claude API"))
print(todo.add("完成项目"))
print(todo.list())
```

## 2.5 实战：完整项目开发

### 项目：AI 助手命令行工具

**功能**:
- 对话交互
- 历史记录
- 多轮对话
- 上下文管理

**完整代码**:
```python
import os
from anthropic import Anthropic

class ClaudeCLI:
    def __init__(self):
        self.client = Anthropic(api_key=os.getenv("ANTHROPIC_API_KEY"))
        self.messages = []
    
    def chat(self, user_input):
        self.messages.append({
            "role": "user",
            "content": user_input
        })
        
        response = self.client.messages.create(
            model="claude-sonnet-4-5-20250929",
            max_tokens=1024,
            messages=self.messages
        )
        
        assistant_message = response.content[0].text
        self.messages.append({
            "role": "assistant",
            "content": assistant_message
        })
        
        return assistant_message
    
    def clear_history(self):
        self.messages = []
        return "历史记录已清除"
    
    def show_history(self):
        if not self.messages:
            return "暂无历史记录"
        for msg in self.messages:
            role = "👤" if msg["role"] == "user" else "🤖"
            print(f"{role}: {msg['content'][:100]}...")

# 使用示例
if __name__ == "__main__":
    cli = ClaudeCLI()
    print("Claude 命令行助手，输入'quit'退出")
    
    while True:
        user_input = input("👤: ")
        if user_input.lower() == "quit":
            break
        response = cli.chat(user_input)
        print(f"🤖: {response}")
```
