# 飞书机器人入门教程

## 什么是飞书机器人？

飞书机器人是一个可以自动回复消息、执行任务的 AI 助手。

## 快速开始

### 步骤 1: 创建应用

1. 访问 https://open.larksuite.com/app
2. 登录飞书开放平台
3. 点击「创建应用」
4. 选择「机器人」类型

### 步骤 2: 配置权限

需要的权限：
- ✅ 发送消息
- ✅ 接收消息
- ✅ 访问群聊

### 步骤 3: 获取凭证

- App ID: `cli_a93e6722b4615bd8`
- App Secret: 在后台查看

### 步骤 4: 启动机器人

```bash
# 使用 feishu-bridge
cd feishu-bridge
npm install
npm start
```

### 步骤 5: 测试

在飞书群里 @机器人，看是否回复。

## 进阶功能

### 自动回复关键词

```javascript
if (message.includes('你好')) {
  reply('你好！有什么可以帮你？');
}
```

### 定时任务

```javascript
// 每天早上 9 点发送日报提醒
cron('0 9 * * *', () => {
  send('该写日报了！');
});
```

### 接入外部 API

```javascript
// 调用天气 API
const weather = await fetch('https://api.weather.com/...');
send(`今天天气：${weather}`);
```

## 常见问题

**Q: 机器人不回复？**
A: 检查是否开启了「接收消息」权限

**Q: 如何发布到群聊？**
A: 在飞书开放平台添加「机器人」到群

**Q: 支持图片吗？**
A: 支持，使用富文本消息类型
