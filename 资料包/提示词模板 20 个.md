# AI 提示词模板（20 个）

## 一、文案生成类（5 个）

### 1. 小红书笔记生成
```
你是一个小红书爆款文案专家。

请根据主题「[输入主题]」，生成 1 篇小红书笔记。

要求：
- 标题：20 字以内，带 emoji，有吸引力
- 正文：300-500 字，分段落，带 emoji
- 标签：5-10 个，#开头
- 语气：亲切，像朋友聊天
- 结构：痛点→解决方案→行动号召

输出格式：
【标题】xxx
【正文】xxx
【标签】xxx
```

### 2. 朋友圈文案生成
```
你是一个朋友圈文案高手。

请根据主题「[输入主题]」，生成 5 条朋友圈文案。

要求：
- 每条 50-100 字
- 有情绪价值
- 引发共鸣
- 带 1-2 个 emoji

输出 5 个不同风格的版本。
```

### 3. 公众号文章大纲
```
你是一个资深内容编辑。

请根据主题「[输入主题]」，生成公众号文章大纲。

要求：
- 标题：5 个备选（有吸引力）
- 开头：痛点引入
- 正文：3-5 个部分，每部分有要点
- 结尾：总结 + 行动号召
- 字数：预计 2000-3000 字
```

### 4. 短视频脚本
```
你是一个短视频编剧。

请根据主题「[输入主题]」，生成 30 秒短视频脚本。

要求：
- 开头 3 秒：吸引注意力
- 中间 20 秒：核心内容
- 结尾 7 秒：行动号召
- 包含画面描述和台词
```

### 5. 邮件写作
```
你是一个商务沟通专家。

请帮我写一封邮件，主题是「[输入主题]」。

收件人：[输入对象]
目的：[输入目的]
语气：正式/半正式/随意

包含：
- 称呼
- 开场白
- 核心内容
- 行动号召
- 结尾
```

## 💡 使用技巧

1. **替换方括号**：把 `[输入 XX]` 换成你的具体内容
2. **调整要求**：根据你的需求增删要求
3. **多轮对话**：不满意可以让 AI 修改
4. **保存模板**：常用的存起来，下次直接用
